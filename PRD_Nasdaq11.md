## 2. 배경 — v10 실측이 가리킨 세 개의 화살표

1. **상태 0 = 미검증 제2 엣지**: 트렌드 템플릿 통과 + 피벗 −10% 이상 눌림 + 점수 상위의 교집합이 +36~40%p(승률 84~91%). 이론 족보: 모멘텀 ∩ 단기 반전(Jegadeesh-Titman / Jegadeesh 1990), 처분 효과 오버행(Grinblatt-Han 2005), 오닐 10주선 눌림목·포켓 피벗·와이코프 재매집. 단 In 구간(+1.1%p)과의 격차 → 국면(V자 회복)의 몫 분리가 필수. 표본 소(11~56건).
2. **청산은 조건부로만 유효**: 상시 가동 5종 전멸. 단서 — 발동률 83.7%(방어선이 정상 호흡 안), 클라이맥스만 중앙값 개선(강세 매도), 하락 국면 중앙값 −10~−24%p(청산이 일할 자리). → 시점·공간·국면·방향의 조건부 청산 설계.
3. **거래량 확인 = 실행 계층의 공용 문법**: trigger_vol 조건부 승률 59.5/64.2% — 단 게이트화는 기회비용으로 부결(미체결 승자 = 상태 0 현상과 동일 뿌리). → 선별(모드)·실행(트랙) 관심사 분리: 모드는 "무엇을", 트랙은 "언제·얼마에", 거래량 확인은 트랙 공용.
4. **약세장 데이터 부재**: 하락 국면 유효 표본 2~5건 — 1~3의 결정적 시험대가 데이터에 없음. 2021년 기점 확장으로 2022 약세장 확보 (10년은 생존 편향 한계비용 체증으로 비경제적).
5. **펀더멘털 허들의 한계효용은 유니버스 품질에 반비례**: NGX에서 허들 Partial 그룹(n=101)이 60일 중앙값 +7.31%p — 복권 구조(전체 중앙값 −0.90)의 NGX에서 거의 유일하게 "전형적 거래가 이기는" 부분집합. NDX에서는 지수 편입 자체가 품질 필터라 허들이 보조적이었으나, 시총 순 100개일 뿐인 NGX에서는 허들이 복권과 적금을 가르는 칼이 됨 — 미너비니 원전(기술+펀더멘털 필수 결합)과 정합. → 교집합(★★∩허들) 검증이 NGX 재판정의 경로.

### 4.2 눌림목 진입 변형 (`pullback_*` 3종 세트)

**관찰 조건 (신호일 평가 — "건강한 눌림" 판별, 상수는 constants/pullback.js):**
```
P1. 트렌드 템플릿 통과 (기존)                          — 장기 구조 온전
P2. 피벗(63일 선행 고점) 대비 −10% ~ −25%             — 눌림 깊이 (T7이 −25% 초과 차단)
P3. 현재가 > SMA200 (기본) — SMA50 엄격형은 상수로만 예비
P4. 눌림 구간(피벗 이후) 평균 거래량 < 직전 상승 구간(피벗 이전 동일 길이) 평균 — 고갈(dry-up)
```

**3종 세트 (확인의 대가를 분리 측정 — confirm2 설계 문법):**
| 변형 | 진입 규칙 | 검증 목적 |
|---|---|---|
| `pullback_immediate` | 관찰 조건 충족 신호일 종가 진입 | 기준선 (상태 0 실측의 재현) |
| `pullback_resume` | 관찰 후 30거래일 내 종가 > 직전 10거래일 최고 종가 시 진입 (체결가 = max(트리거, 시가)) | 재개 확인의 순효과 |
| `pullback_resume_vol` | 위 + 당일 거래량 ≥ 1.5×50일 평균 | 거래량 문법의 추가 가치 (발견 #8 이식) |

- 미체결 별도 집계 + 체결률 + 두 벌 성과(조건부/기회비용 포함) — entryVariants 문법 그대로
- 손절(청산 C 결합): 눌림 저점(관찰 시점까지의 피벗 이후 최저 종가) × 0.98 — "가설 파기" 구조 손절
- 집계: allSignals 기준 필수(표본 확보), top5 병행, **국면별 분해 필수** (상승/중립/하락 — V자 국면 의존성 검증)

### 4.3 청산 변형 A·C·E (+F는 §3-3)

| 후보 | 규칙 | 근거 발견 |
|---|---|---|
| **A. 국면 조건부** | 신호일 국면 상승 → 60일 보유 / 신호일 국면 중립·하락 → ATR 손절(2.5×ATR14) 가동. (Should) regime-flip: 보유 중 국면이 하락 전환 시 당일 종가 청산 | #10 + #2 (청산이 일할 자리 = 하락 국면) |
| **C. 구조 기반 손절** | 돌파형 진입: 피벗 × 0.97 이탈 시 청산 / 눌림목형: 눌림 저점 × 0.98 — 진입가 대비 고정 %가 아닌 가격 구조 기준 | #4 (미너비니 원전의 실제 방식 — 손절은 구조 이탈 거리) |
| **E. 클라이맥스 부분 청산** | X4(클라이맥스 런) 발생 시 50% 청산·50%는 60일 보유 — 수익률 = 가중 평균 | #10 (유일한 중앙값 개선 규칙의 트레이드오프 절충) |

- 전부 종가 기준 판정 한계 문구 고정, adopted=false, 채택은 §5 기준 + 운영자
- 대표 조합 2개 이내: `pullback_resume_vol × C(눌림 저점)`, `entry_close × A`

### 4.4 허들 교집합 집계 (양 유니버스 — NGX 재판정 + 보편성 검증)
- **양 유니버스 공통**으로 교집합 축 추가: ★★ 컨센서스 ∩ 펀더멘털 (Pass / Partial 이상 / Fail) × byHolding × 국면 — **중앙값이 1차 지표**
- NGX: 복권 구조의 해독 여부 판정 (재판정 재료) / **NDX: 비교 기준** — 교집합 효과가 저품질 유니버스 특유인지(가설: NGX에서 개선 폭이 큼) 보편인지 대조. NDX에서도 교집합이 유의미하게 우수하면 "허들 게이트 강화"가 유니버스 무관 개선 후보로 승격
- 펀더멘털 축의 기존 한계 유지: 근사 재구성·짧은 커버리지(coveredFrom) 참고치 — 교집합 결과에도 동일 라벨 부착
- 재판정 기준(선커밋, §5): NGX 통과 시 "허들 게이트 조건부 NGX 노출"을 v12 검토 안건으로

### 4.5 집계 확장 (schemaVersion 4)
- `stateRegimeAxis`: 진입 상태(0~3) × 국면 × 전략 × sample — 상태 0의 구조 몫 vs 국면 몫 분리
- `entryVariants`에 strategyKey 축 추가: trend / minervini / consensus_2star 풀 각각에서 진입 변형 성과 (② — 미너비니 모드에서 거래량 돌파 효과의 공백 해소)
- `pullbackAxis`: 3종 세트 + 국면 분해 결과
- 검증기 v4 + 로더 v1~v3 하위 호환 (기존 화면 무변화)

### 4.6 relax_off_in_downturn 앱 반영 (사상 첫 채택)

- constants changelog 필수 기재: 날짜 · 근거 리포트(v10 backtest) · **"하락 국면 Out 표본 박약 — 분기 재검증 시 재확인" 조건**
